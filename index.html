Paso 2: Crear tu USB de Instalación con Rufus

Rufus es la herramienta por excelencia para crear unidades USB de arranque. Tomará la imagen ISO que descargaste y la preparará en una memoria USB, convirtiéndola en el medio de instalación de Windows 11.

Cómo usar Rufus:

  1. Descarga la última versión de Rufus desde su página oficial (la versión portable es ideal).
  2. Inserta una memoria USB de al menos 8 GB. ¡Atención! Este proceso borrará todos los datos del USB.
  3. Abre Rufus. En "Dispositivo", asegúrate de que esté seleccionada tu memoria USB.
  4. Haz clic en "SELECCIONAR" y elige el archivo ISO de Windows 11 que descargaste.
  5. Confirma que el "Esquema de partición" sea GPT y el "Sistema de destino" sea UEFI (no CSM).
  6. Presiona "EMPEZAR". Aparecerá una ventana crucial: aquí puedes eliminar los requisitos de TPM 2.0, Secure Boot y 4GB de RAM, ideal para PCs más antiguos.
  7. Acepta la advertencia final y espera a que el proceso termine. Cuando la barra se ponga verde y diga "PREPARADO", tu USB estará listo.

Paso 5: Pasos Finales y Recomendaciones

Instalar Microsoft Store (Opcional)

Si tu versión de Windows no incluye la Store, puedes instalarla manualmente. Abre PowerShell como Administrador y ejecuta este comando:

Get-AppXPackage -AllUsers -Name Microsoft.WindowsStore | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml" -Verbose}

Instalar Drivers del Fabricante

Visita la web del fabricante de tu placa base (ASUS, Gigabyte, MSI, etc.) y descarga los drivers más recientes, especialmente los del Chipset. Esto es crucial para la estabilidad y rendimiento del sistema.

Ejecutar Windows Update

Una vez tengas los drivers principales, ve a Configuración > Windows Update y busca actualizaciones. Esto instalará parches de seguridad importantes y puede que encuentre drivers de otros componentes.
